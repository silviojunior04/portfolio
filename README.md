# SJdev Portfolio

Portfólio pessoal de Silvio Paulo, com foco em desenvolvimento web e diferencial em infraestrutura, Linux, redes e monitoramento.

## Arquivos

- `index.html` — estrutura HTML do site
- `style.css` — estilos visuais da identidade SJdev
- `script.js` — scripts do menu mobile e ano automático
- `assets/logo-sjdev.svg` — logo vetorizada para perfil/avatar
- `assets/banner-linkedin-sjdev.svg` — capa vetorizada para LinkedIn

## Publicação no Cloudflare Pages

1. Envie o arquivo `index.html` para seu repositório ou faça upload direto.
2. Configure o domínio `sjdev.com.br` no Cloudflare.
3. Ajuste os links de GitHub e LinkedIn dentro da seção de contato.
